[ 01 ]  Deploy

Clone-free on Mac & Linux.

The one-liner fetches install.sh, which inspects your machine, adopts your provider and keys, and verifies the whole stack before it hands you the dashboard.

  • DETECTSYour machinePython 3.11+, a browser, an existing Hermes install — it picks the right path automatically.
  • ADOPTSYour provider40+ LLM providers via hermes setup, including any custom / OpenAI-compatible endpoint. Already configured? It reuses your keys and skips the wizard.
  • VERIFIESThe stackhermes-swarm doctor confirms runtime, model reachability, Chromium, and the Hermes compat seams before you start.

From zero to a working team in three steps.

01

Configure a provider

One hermes setup wizard — pick from 40+ providers (or a custom / OpenAI-compatible endpoint), paste a key, choose a model. The one-liner runs it for you.

~ 2 MINUTES
02

Build a team

Tell the Architect what you want to accomplish. It proposes the agents, their roles, and who talks to whom — then builds it live on your approval.

GUIDED BY THE ARCHITECT

RUNS UNATTENDED
[ 03 ]  Self-host

Your hardware. Your rules.

No SaaS, no tenant, no data leaving your box. Run it on a laptop for an afternoon or a VPS for a year — the specifications are the same.

Deploy on a VPS
RuntimeOne line, or Docker. The local install builds a Python 3.11+ venv; the Docker image bundles Python, Hermes, Chromium and the dashboard, with agents contained inside it.
Providers40+ LLM providers via hermes setup — Anthropic, OpenAI, OpenRouter, Groq, DeepSeek, local models — including any custom / OpenAI-compatible endpoint (pick the custom provider and enter its URL + key).
Access controlSingle shared API key. Set SWARM_API_KEY and every endpoint and the live WebSocket require it; the dashboard prompts once.
StateEverything on disk under one data dir — team configs, task queues, conversation history, the shared project, and credentials. Rotating backups on every save.
Observability/health + structured logs. Uptime, queue depth, and backend reachability for any uptime monitor; logs to stdout or a rotating file.
LicenseOpen source. Self-hostable, inspectable, yours to fork.
